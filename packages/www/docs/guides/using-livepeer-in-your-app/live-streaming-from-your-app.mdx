# Live streaming from your app

There are many 3rd party encoders that allow streamers to push RTMP to the
Livepeer.com global-optimized ingest URL is `rtmp://rtmp.livepeer.com/live/`.
These include [OBS](https://obsproject.com/), [ffmpeg](https://ffmpeg.org/),
[Larix](https://softvelum.com/larix/) for iOS and Android, Streamlabs OBS for
[Desktop](https://streamlabs.com/),
[Android](https://play.google.com/store/apps/details?id=com.streamlabs&hl=en_US&gl=US),
and
[iOS](https://apps.apple.com/us/app/streamlabs-live-streaming-app/id1294578643).
You can also use any RTMP library, for example
[Streamaxia](https://www.streamaxia.com/) in iOS.

Livepeer.com is currently working on additional recommendations for mobile SDKs.
Fill out a [contact form](contact) if you have more urgent needs.
