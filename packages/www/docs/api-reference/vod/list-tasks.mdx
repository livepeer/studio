# Tasks

You can track the status of an import or an export Task through the Livepeer
API.

# List tasks

List all tasks in the account, most recent first.

```jsx
curl --location --request GET 'https://livepeer.com/api/task' \
--header 'Authorization: Bearer $API_TOKEN'
```

# Retrieve a Task

Retrieves the details of a `Task`, including progress updates and status. Supply
the unique `Task` ID that was returned from your previous request or from an
`Asset` import or export.

```jsx
curl --location --request GET 'https://livepeer.com/api/task/$TASK_ID' \
--header 'Authorization: Bearer $API_TOKEN'
```
